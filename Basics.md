- In python we don't have `nul`. We use a case sensitive value called None 
- For creating objects in python we don't have a new keyword. We can simply call class name like a function name 
	eg: result = ListNode(...) 

- Like we have paranthesis in other languages we don't use it python. It doesn't crash anything. But it is not the convention that people use 
	Eg: if l1.val + l2.val > 9: 
- Instead of `==` and `!=` we should use is and is not
